Features

Built for people who
copy things all day

Not just another clipboard manager. L2cache understands what you copy.

🧠
AI-generated titles
Code clips show what the code does, not what language it's in. "Authenticate user and store session in Keychain" — not "Swift function".
⚡
Smart Paster
Hold ⌥V instead of ⌘V. Transform your clipboard before pasting — strip markdown fences, fix grammar, convert code, clean URLs, format JSON.
🍎
Apple Intelligence
Runs on-device using Apple's foundation model. Private by default — your clipboard never leaves your Mac unless you explicitly enable cloud AI.
🔍
Search everything
Full-text search across all clips. Screenshots are searchable by their text content via OCR. Find anything in under 100ms.
🛡
Sensitive radar
API keys, passwords, and credit cards are automatically detected and never stored. A shield badge confirms when something was blocked.
📦
Smart bundles
Everything you copy during a coding session groups automatically. "Auth Sprint — 12 clips" — recall an entire context with one click.
Who it's for

Three kinds of power users

If you copy more than 20 things a day, L2cache was built for you.

🤖
AI Engineers & Developers
Juggling LLM outputs, stack traces, and prompts across Windsurf, Cursor, and ChatGPT. L2cache strips markdown fences, isolates error messages, and never lets a good prompt disappear.
✍️
Researchers & Writers
Copying quotes and URLs from 40 browser tabs. L2cache enriches every URL with its page title, strips tracking parameters, and groups clips by topic automatically.
📊
Data & Ops Professionals
Copying tables from PDFs and emails into Excel — only to watch the formatting break. L2cache detects tabular data and offers clean CSV or markdown table in one click.
