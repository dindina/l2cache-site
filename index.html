Every copy saved, understood, and ready when you need it.

🧠
AI-generated titles
Code clips show what the code does, not what language it's in. "Authenticate user and store session in Keychain" — not "Swift function".Requires macOS 15.1+ and Apple Silicon. Falls back to plain-text titles on other Macs.
⚡
Smart Transform
Copy smarter. Transform your clipboard as you copy — strip markdown fences, fix grammar, convert code, clean URLs, format JSON.AI-powered transforms require macOS 15.1+ and Apple Silicon. Local instant transforms work on any supported Mac.
🍎
Apple Intelligence
Runs on-device using Apple's foundation model. Private by default — your clipboard never leaves your Mac.Apple Intelligence requires macOS 15.1+ and Apple Silicon. Core clipboard features work on macOS 13+ regardless.
🔍
Search everything
Full-text search across all clips. Screenshots are searchable by their text content via OCR. Find anything in under 100ms.
🛡
Sensitive radar
API keys, credit card numbers, private keys, SSNs, and WiFi credentials are automatically detected and stored as hidden. Content stays locked — unlock with Touch ID or your password to copy it.
🗂
Smart albums
Clips are automatically grouped into albums by type and context — code, URLs, images, and more. Browse by album to find related clips instantly.
📱
QR code generation
Copy any URL, text, or contact detail and generate a QR code in one click. Share instantly without any extra app.
🖼
Screenshot renaming
Screenshots are automatically renamed using AI — no more "Screenshot 2025-06-18 at 10.34.02". Find your screenshots by what's actually in them.
